# Static deploy (Vercel + GitHub)

This is a static version of your ESP32 dashboard UI.

## Deploy
1. Push this folder to GitHub
2. Import the repo in Vercel (Framework: Other)
3. No build command needed

## Notes
- MQTT uses the public CDN: https://unpkg.com/mqtt/dist/mqtt.min.js
- The login page is a static mock at /login/
